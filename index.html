Documentation

Overview:

This Software/ App is able to extracts Id and Class from from html document. You can select file manually, drag and drop a file or you can simply paste your code in the paste area and click proced. It takes a little time to do the task.

Use of this App

This app is intented to give benefit to frontend devlopers. Mostly who uses HTML and JavaScript to build their applicaation. This app will save their time by automatically extracting id's under :> document.getElementById("id"); and class under :> document.querySelector(".class");

How does this app work?

You upload your desired file and choose whether you want the id's or class'es only. Then it process it's function and in the output section it prints the result.

Does this app collect, log or store data?

NO. This app doesn't collect, store or log any user input.

Is there any scope of developing this app further?

Yes. If you're someone who can make extension for offline/online code editor like "VS Code" and for similar platform, then you're welcome to work on this.
